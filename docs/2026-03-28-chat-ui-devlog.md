# 2026-03-28 Chat UI 开发记录

## 基线说明
- 对比基线：`origin/feature/chat-ui-upgrades`
- 记录范围：截至今日在本地分支上完成并保留的 UI 改动，以及当天尝试后已回退的实验项

## 今日完成的主要内容

### 1. 左侧侧栏整体视觉重构
- 侧栏顶部整体上移，收窄了标题区上方净空。
- `助手 / 话题 / 设置` 标签页改为更紧凑的横向 tab 结构，并补充了更完整的无障碍属性。
- `VCP Agents` 标题区改成伪列表项风格，整体尺寸、边距和列表节奏对齐。
- `搜索助手或群` 搜索框改为与列表项更接近的胶囊式布局，加入左侧搜索图标占位。

### 2. 助手列表 / 话题列表胶囊样式统一
- 左侧列表项统一为更紧凑的胶囊尺寸：
  - 列表项高度统一为 `37px`
  - 外层横向留白收紧
  - 头像、标题、计数徽标整体缩放到更紧凑的密度
- 话题列表选中态改为更稳定的玻璃高亮表现。
- 列表 hover 改成较弱的玻璃质感，避免过白过亮。

### 3. 搜索框与列表项的尺寸、间距对齐
- 搜索框与下方列表项统一高度、左右边距、内部 padding。
- 搜索框与列表的垂直间距调成与列表项之间的节奏接近。
- 搜索框默认态去掉外层液态玻璃框，hover / focus 仍保留强化表现。

### 4. 侧栏底部操作按钮液态玻璃化
- `创建Agent` / `创建 Group` 两个按钮改为液态玻璃样式。
- 为浅色与深色模式分别提供不同强度的玻璃参数。

### 5. 主聊天区顶部栏压缩
- `chat-header` 恢复为实色 / 毛玻璃底板方案。
- 在此基础上继续收窄顶部栏高度：
  - 顶部 padding 从更厚版本收窄到当前紧凑版本
  - 标题字号缩小
- 左侧侧栏顶部净空也同步收窄，使主界面上下密度更统一。

### 6. 聊天区右上角按钮样式微调
- 顶部按钮默认态加入轻微玻璃边框与细弱玻璃底。
- 浅色模式 hover 改为明确的主题色背景，不再出现发白问题。
- `toggleAssistant` 在浅色模式下的选中态保持默认底色，但边框改为红色强调。
- 按钮圆角加大到当前版本，保留统一的圆角矩形形态。

### 7. 第四个按钮（Create Topic）定制化
- 将第 4 个按钮改为定制组合 SVG，不再显示右侧独立文案。
- 组合 SVG 最终方案：
  - 左侧为聊天框 `+` 图标
  - 右侧为两行 `CREATE / TOPIC`
- 按钮显示逻辑从 `inline-block` 改为 `inline-flex`，确保图标与文案布局稳定。
- 对 SVG 在按钮内的尺寸和水平位置进行了微调，当前版本保留最终校正结果。

### 8. 侧栏标签页无障碍与键盘导航
- 为侧栏 tab 和 tabpanel 增加 `role` / `aria-*` 属性。
- 支持 `ArrowLeft` / `ArrowRight` / `Home` / `End` 在 tab 间切换。

### 9. 主题与样式基础调整
- `styles/themes.css` 当前相对远程有一组较大的主题定义变更，主体方向为切换到 EVA 风格主题变量。
- 这部分属于今日工作树的一部分，随同 UI 样式调整一起纳入本次提交。
