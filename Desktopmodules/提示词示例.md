## 桌面挂件 API（VCPdesktop Widget Sandbox）

当你需要在桌面画布上创建交互式挂件时，使用 <<<[DESKTOP_PUSH]>>> 语法输出 HTML/CSS/JS。
你的脚本在沙箱中执行，以下 API 自动可用：

### document（已代理到 Shadow DOM）
- document.querySelector/querySelectorAll/getElementById → 在 widget 内部查找
- document.createElement/createTextNode → 正常创建元素

### vcpAPI（后端数据访问，自动认证）
- vcpAPI.weather() → 获取天气数据
- vcpAPI.fetch('/admin_api/任意端点') → 通用后端 API

天气数据结构：
{
  city, hourly: [{fxTime, temp, text, icon, humidity, windDir, windScale}],
  daily: [{fxDate, tempMin, tempMax, textDay, iconDay}],
  airQuality: {aqi, category, pm2p5},
  warning: [{title, text}]
}

### musicAPI（音乐播放器控制）
- musicAPI.getState() → 获取播放状态
  返回: {is_playing, file_path, position_secs, duration_secs, volume, ...}
- musicAPI.play() → 播放
- musicAPI.pause() → 暂停  
- musicAPI.seek(秒数) → 跳转
- musicAPI.setVolume(0-100) → 设置音量
- musicAPI.send('music-remote-command', 'next') → 下一首
- musicAPI.send('music-remote-command', 'previous') → 上一首

### 使用示例

天气挂件：
<<<[DESKTOP_PUSH]>>>
<div id="w" style="padding:16px;background:rgba(0,0,0,0.5);color:#fff;border-radius:12px;">加载中...</div>
<script>
vcpAPI.weather().then(function(data) {
    var h = data.hourly[0];
    document.getElementById('w').innerHTML = '<h2>'+h.temp+'°C '+h.text+'</h2><p>湿度'+h.humidity+'%</p>';
});
</script>
<<<[DESKTOP_PUSH_END]>>>

音乐控制条：
<<<[DESKTOP_PUSH]>>>
<div style="display:flex;gap:8px;padding:12px;background:rgba(20,20,40,0.85);border-radius:20px;color:#fff;">
    <span id="title">--</span>
    <button onclick="musicAPI.send('music-remote-command','previous')">⏮</button>
    <button id="pp" onclick="this.dataset.playing==='true'?musicAPI.pause():musicAPI.play()">▶</button>
    <button onclick="musicAPI.send('music-remote-command','next')">⏭</button>
</div>
<script>
setInterval(async function(){
    var s = await musicAPI.getState();
    if(!s) return;
    var pp = document.getElementById('pp');
    pp.textContent = s.is_playing ? '⏸' : '▶';
    pp.dataset.playing = s.is_playing;
    var fp = s.file_path || '';
    var parts = fp.split(/[\\\\/]/);
    var name = parts[parts.length-1] || '--';
    var dot = name.lastIndexOf('.');
    if(dot>0) name = name.substring(0,dot);
    document.getElementById('title').textContent = name;
}, 2000);
</script>
<<<[DESKTOP_PUSH_END]>>>
